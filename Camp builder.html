One-time Supabase setup

The database table doesn't exist yet. This takes about 60 seconds to fix:

  1. Go to supabase.com → your project → SQL Editor (left sidebar)
  2. Paste and run this SQL:
CREATE TABLE IF NOT EXISTS lodge_state ( id INTEGER PRIMARY KEY DEFAULT 1, data JSONB NOT NULL DEFAULT '{}', updated_at TIMESTAMPTZ DEFAULT NOW() ); INSERT INTO lodge_state (id, data) VALUES (1, '{}') ON CONFLICT (id) DO NOTHING; ALTER TABLE lodge_state ENABLE ROW LEVEL SECURITY; CREATE POLICY "allow_all" ON lodge_state FOR ALL USING (true) WITH CHECK (true);
  1. Also go to Database → Replication and enable replication for the lodge_state table so live sync works
  2. Come back here and click Retry

Welcome to Camp Builder
Camp Builder is your hub for managing multiple lodge construction projects.

→ Create a camp using the + card — name it, pick a location and colour.
→ Open a camp by clicking its card to enter the full project manager.
→ Each camp has its own rooms, orders, suppliers, snags and notes — completely separate.
→ The summary stats on each card update live so you can see project health at a glance.

Project Manager
Everything for one camp, in one place.

→ Rooms — add spaces, log L×W×H, then hit Calculate materials for quantities.
→ Orders — track every item Quote → Ordered → In transit → Arrived. Add photos on delivery.
→ Suppliers — store contacts with transit times. WhatsApp or email them directly.
→ Snag list — punch list of issues before handover. Photo-document problems.
→ + button (bottom right) — fastest way to log anything on site.
